# Visão Geral do Game Design
Jogo top-down em que o jogador deve arremessar pedras em um lago para acertar balões espalhados pelo cenário controlando a direção e força do arremesso, neste cenário há obstáculos espalhados com diferentes efeitos sobre a pedra lançada, podendo ser por exemplo: mudança de trajetória, diminuição ou aumento das velocidades, multiplicar pedras, etc. Ao estourar todos os balões a fase é finalizada e a pontuação final é contabilizada.

# Público Alvo
Livre
Jogadores casuais que utilizam smartphone e querem um jogo para passar o tempo
